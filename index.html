Family Portfolio Management

A private personal finance app for tracking bank accounts, transactions, and investment portfolios.

About

Family Portfolio Management (FinLAN) is a self-hosted personal finance application designed for private home network use. It allows a single household to securely monitor bank accounts, credit cards, and investment portfolios in one place.

The application uses the Plaid API solely to retrieve financial data on behalf of the account owner. No data is shared with third parties, sold, or used for advertising purposes.

How We Use Your Data

  • Financial data is accessed only by the account owner.
  • Data is stored locally on a private home server — never in the cloud.
  • No data is shared with or sold to any third party.
  • Bank connections are used only to read account balances and transactions.
  • Credentials are never stored; authentication is handled entirely by Plaid.

Privacy

This is a single-user, private application. All financial data remains within the user's local network. We do not collect, transmit, or store any personally identifiable information on external servers.

Bank account connections are managed through Plaid's secure OAuth flow. Users can revoke access at any time via the Plaid Portal.
